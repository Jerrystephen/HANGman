# Hangman Game

So this a very simple and interactive Python implementation of the classic word-guessing game Hangman. This pretty much allows players to guess letters or words to uncover a hidden word while avoiding a hanging fate!

## Game Features

- Randomly selects a word from a predefined list.
- Supports guessing by both letters and whole words.
- Visual feedback of the hangman drawing as incorrect guesses accumulate.
- Tracks previously guessed letters and words to avoid repetition.
- Allows replaying the game after a win or loss.
